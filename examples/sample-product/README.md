## Product Summary

**TaskFlow** helps small teams (5-15 people) manage tasks without the complexity of enterprise project management tools.

- **Problem**: Existing tools are either too simple (just todo lists) or too complex (Jira, Asana)
- **Solution**: A "Goldilocks" task manager that's just right for small teams
- **Target Users**: Startup teams, small agencies, freelancer teams

## What This Example Demonstrates

1. **Discovery Phase**: How to document a problem with evidence
2. **Definition Phase**: How to write user stories with acceptance criteria
3. **Architecture Phase**: How to structure technical decisions
4. **Execution Phase**: How to break work into AI-executable tasks

Use these documents as templates for your own products!
